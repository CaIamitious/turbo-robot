# Icon generation

Replace the placeholder icons in this folder with real PNG icons before deploying.

Required files:
- `icon-192.png` — 192×192px
- `icon-512.png` — 512×512px
- `apple-touch-icon.png` — 180×180px (copy of icon-192 or dedicated version)

Free tools to generate all sizes from a single image:
- https://realfavicongenerator.net
- https://maskable.app (for maskable icon preview)
